# Research documents

Follow the canonical research process — tiering (Quick Note vs Full Research), naming
(`Research/<topic>-<type>.md`), the header with Status + Confidence, evidence-vs-interpretation
separation, and the register-in-CLAUDE.md step (the most-skipped, most valuable one):

➡️ **[`Research/RESEARCH-PROCESS.md`](../../../Research/RESEARCH-PROCESS.md)**

That file is the single source of truth — edit it there, not here. Check the existing research table /
`Research/` folder **first**; the work may already exist.
